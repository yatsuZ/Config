# Rapelle

## Pour push

```sh
git init

git add .

git commit -m "first commit"

git branch -M main

git remote add origin <lien repo>

git push -u origin main
```

- ln -s permet de faire des lien, et de redirection.
- xdg-open permet douvrire un explorateur de fichier TRES UTILE.
